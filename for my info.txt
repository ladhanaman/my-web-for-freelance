  1. GridBackground
  Magnetic Elastic Grid (or Interactive Mesh Distortion)                                                                                                       
  Grid intersection points are spring-physics particles pulled toward the cursor. The pattern — particles with velocity, spring stiffness, and damping — is
  called a spring-mass system. Common in creative agency sites.                                                                                                
                                                                                                                                                               
  ---                                                                                                                                                          
  2. HeroSection canvas                                                                                                                                        
  Grid-snapped Cursor Trail (or Pixelated Mouse Trail)      
  Broader category: Cursor Particle Trail. The grid-snapping gives it the pixel art aesthetic. The time-based fade using performance.now() is standard in
  canvas-based trail effects.
                                                                                                                                                               
  ---
  3. ScrambleToggle                                                                                                                                            
  Text Scramble Effect (also called Hacker Text, Glitch Text, or Caesar Scramble)
  Popularised by sites like Linear.app and Vercel. The requestAnimationFrame-driven random character swap before resolving to the target string is the
  industry-standard implementation.                                                                                                                            
                                                                                                                                                               
  ---                                                                                                                                                          
  4. GunHero — overall section                                                                                                                                 
  Scroll-Driven 3D Scene using a Sticky Scroll container                                                                                                       
  The 200vh outer / 100vh sticky inner pattern is called a scroll canvas or scrollytelling section. The spring interpolation (1 - Math.exp(-stiffness * delta))
   is exponential smoothing, the standard for frame-rate-independent spring physics in 3D.                                                                     
                                                                                                                                                               
  ---                                                                                                                                                          
  5. Framescape floating cards                                                                                                                                 
  Cursor Parallax (or Mouse Parallax with Depth Layers)                                                                                                        
  Each card has a depth value controlling how much it moves — this layered movement creates a parallax depth illusion. Sometimes called tilt parallax or
  perspective parallax.                                                                                                                                        
                                                                                                                                                               
  ---
  6. Framescape scroll lock                                                                                                                                    
  Scroll Hijacking (sometimes politely called Scroll Snapping or Viewport Lock)
  Deliberately taking control of the scroll for a set duration. Used by Apple product pages heavily. Controversial UX pattern — the industry debates it.
                                                                                                                                                               
  ---                                                                                                                                                          
  7. Gradient orbs in HeroSection                                                                                                                              
  Gradient Blobs (also Aurora Background, Mesh Gradient Orbs, or Bokeh Ambient Lights)                                                                         
  The radial-gradient + blur + CSS animation combination is called a glassmorphism ambient background or simply gradient blobs. Huge in 2022–2024 design
  trends.                                                                                                                                                      
                                                                                                                                                               
  ---                                                                                                                                                          
  8. NavBar hide/show on scroll                                                                                                                                
  Smart Navbar or Scroll-Aware Navigation (technically Hide-on-Scroll Header)                                                                                  
  The pattern of hiding on scroll-down, showing on scroll-up is called the auto-hide navbar pattern. First widely used by Medium.
                                                                                                                                                               
  ---                                                                                                                                                          
  9. KairosFooter reveal                                                                                                                                       
  Sticky Footer Reveal (or Scroll-Behind Footer, Layered Scroll Reveal)                                                                                        
  The main content scrolling over a pinned footer is a variant of the parallax layering technique — Z-axis parallax via z-index stacking rather than transform.
                                                                                                                                                               
  ---                                                                                                                                                          
  10. Framescape / contact anim-fade-up
  Scroll-Triggered Entrance Animation (or Reveal on Scroll)
  The IntersectionObserver firing a fade + translateY is the modern standard — formerly done with libraries like AOS (Animate On Scroll) or ScrollMagic.

  11. Gallery page — photo cards
  Draggable Scatter Board (or Freeform Card Throw / Interactive Polaroid Scatter)
  Cards start as a seeded-random pile (overlapping stack with varied rotation — called a Fanned Card Stack or Riffle Stack illusion). The user drags them to
  scatter freely across the viewport — this free-placement interaction is called a Drag-to-Scatter or Physics-Adjacent Drag Board. The pattern appears on
  creative agency sites and tools like Milanote/Miro. Positions and z-order are persisted in localStorage so the board is restored exactly on return visits.

  ---

  12. ShaderBackground — contact form submit transition
  WebGL2 GLSL Fragment Shader Transition (Warp / Nebula Scene)
  A fullscreen canvas running a raw GLSL fragment shader that generates a procedural nebula using FBM (Fractional Brownian Motion) layered noise, with a
  CRT-style glitch intro (horizontal hold snap, color bleed) that cleans up into a smooth nebula over ~5 seconds. Triggered on form submit; fires
  onWarpComplete() when the animation ends, handing control back to React. This class of effect is called a Shader Takeover or GLSL Transition Screen —
  common in game engines and high-end creative studios. Frame-rate-independent via performance.now() elapsed time uniform.

  ---

  13. AnimatedGradientBackground — radial gradient breathing background
  Animated Radial Gradient Background (or Breathing Mesh Gradient)
  A requestAnimationFrame loop mutates a radial-gradient CSS property on a div to create a slow, oscillating "breathing" effect. The gradient radius
  expands and contracts within a configurable breathingRange. The pattern is called a Breathing Gradient or Pulsing Mesh Background — a calmer, CSS-only
  alternative to WebGL gradient blobs. Used on the SpiralAnimation loading screen.

  ---

  14. SpiralAnimation — galaxy loading screen canvas
  3D Spiral Galaxy with Camera Fly-Through (Canvas + GSAP Timeline)
  1800 stars are placed along an Archimedean spiral in 3D space using a seeded deterministic RNG (so the galaxy looks identical every run). A GSAP timeline
  drives a virtual camera flying toward the spiral core over 20 seconds on a looping timeline. Stars project from 3D to 2D via perspective divide
  (x_screen = zoom * x_world / depth). Each star's displacement uses a three-phase elastic easing (linear → power → easeOutElastic) to produce a
  snapping bloom as the camera passes. A dot trail follows the spiral path frame-by-frame with alternating left/right rotation — the overall pattern is
  called a Scroll-Free Cinematic Canvas or Procedural Galaxy Fly-Through.

  ---

  15. PixelDog — cursor-following pixel pet
  Cursor-Following Pixel Art Pet with Finite State Machine (Canvas Sprite Animation)
  A 16×14 pixel corgi drawn entirely from hand-coded color arrays, rendered to a canvas with per-frame pixel fill. Follows the cursor using exponential
  lerp with three speed thresholds (idle / walk / run), each triggering its own 4-frame walk/gallop cycle. States: idle → sit → stand (on a timer),
  walk, run, jump (click-triggered with gravity). Squash-and-stretch scales the canvas transform based on velocity magnitude. Adds a paw SVG as the
  page cursor while mounted. This pattern is called a DOM Virtual Pet or Cursor Companion — popularised by the original neko.exe and oneko.js.

  ---

  16. OnekoCat — animated sprite pet inside contact card
  Sprite-Sheet Pet with Mood-Driven Animation and Speech Bubble (Cursor Follower)
  Renders a cat from a sprite sheet (oneko-cat-chat.png) by offsetting background-position per frame — the classic CSS sprite animation technique.
  Follows the pointer within its bounding card using requestAnimationFrame. Supports 16 named sprite sets (directional movement N/NE/E… + idle/talk/
  thinking/happy/success/error/sleeping) driven by an external mood prop. A speech bubble floats above the cat, clamped to stay inside the card. Idle
  → sleeping transition fires after random inactivity. This is called a Sprite-Based DOM Pet or Mood-Driven Character Widget.

  ---

  17. ContactSection — multi-phase contact flow
  State-Machine Multi-Phase Form Flow (Form → Shader Transition → Cat Reveal)
  Three phases ("form" / "shader" / "cat") managed in React state. On submit: a CSS vignette closes over 1.2 s (giving the GPU time to compile the GLSL
  shader), then ShaderBackground mounts and the vignette fades out on the compositor thread so shader compilation on the main thread doesn't block the
  animation. When the shader fires onWarpComplete, the section transitions to a fullscreen cat-reveal. The cat's mood prop is derived from form
  completion signals via a priority-ordered useMemo. A speech queue (speechQueueRef) serialises cat dialogue — preventing overlap, deduplicating by key,
  and capping the queue at 2. This orchestration pattern is called a Sequenced Multi-Step UI or Narrative Transition Flow.

  ---

  18. ScrambleHover — hover-triggered text scramble
  Hover-Triggered Character Scramble with Bidirectional Reveal (Text Scramble on Hover)
  Identical scramble mechanic to ScrambleToggle (#3) but triggered by pointer hover, not a boolean toggle. Adds a sequential reveal mode — characters
  resolve one-by-one in configurable direction (start / end / center) — and a reverse animation on hover-exit that un-reveals chars in the opposite
  order. Revealed indices are tracked in a useRef Set (not state) to avoid re-triggering the effect. Pattern: Hover Scramble or Interactive Hacker Text.

  ---

  19. ProgressTracker — contact form completion bar
  Animated CSS Progress Bar with Gradient Glow
  A simple ratio-based progress bar (completed/total → percentage width) driven by a CSS transition-all on the bar's width. Color and box-shadow swap
  from terracotta to green when the bar hits 100 %. The glowing bar is a standard Completion Progress Indicator — the glow is a box-shadow technique
  called Inner Glow or Neon Bar used commonly in dark-theme dashboards.

  ---

  20. DirectEmailBox — contact sidebar email widget
  Copy-to-Clipboard Email CTA with Visual Feedback
  A static card showing the email address as a Gmail deep-link (mailto: equivalent via URL params). A copy button calls navigator.clipboard.writeText()
  and flips to a "Copied!" confirmation state for 2 s before reverting. This is a standard Click-to-Copy pattern — common in developer tool UIs and
  contact cards.
